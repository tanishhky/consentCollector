# Consent Collector

An interactive front-end web application developed during the IUDX (India Urban Data Exchange) Internship. This interface is designed for robust data consent collection flows.

![Consent Collector UI Preview]([Insert Image/Result Here])

## Overview

Modern data exchanges require transparent, clear, and legally compliant consent mechanisms from users before data collection or sharing. This "Consent Collector" interface serves as a functional mock-up and active front-end designed to easily plug into data exchange gateways to manage authorization logic, policies, and privacy protocols.

## Key Components

- **Vanilla Web Technologies**: Built elegantly with standard HTML5, CSS3, and JavaScript, ensuring rapid load times and compatibility across environments without bloating the application with heavy frameworks.
- **Form Workflows**: Multi-step workflows engineered to parse diverse data request types and accurately capture the user's explicit consent parameters.
- **Clear Styling**: Leverages a robust structural css implementation that conforms with professional accessibility guidelines for data-collection forms.

## Repository Structure

```
├── index.html        # Main entry point for the form and application UI
├── Summary.html      # Post-consent summary and review page
├── css/              # Stylesheets for responsive design
└── js/               # JavaScript logic for form validation and state management
```

## Quick Start

Since this is a vanilla HTML/JS project, no heavy dependency installation or build step is required. 

To view and test the interface:
1. Clone the repository.
2. Directly open `index.html` in any modern web browser.
3. Interact with the form flows to navigate to the `Summary.html` page.
